# The Grid System — Bass Theory Trainer

An interactive training tool for learning bass guitar theory using **The Grid System** — a coordinate-based approach to understanding the fretboard.

Instead of traditional music theory framing, everything is taught through positions (string + fret), distances from Home, and genre-specific maps.

## How It Works

- **Positions**: Every point on the bass neck has an address. `A5` = 5th fret on the A string.
- **Home**: Any position can be Home. When someone says "we're in D," find D on the neck and that's Home.
- **Distances**: From Home, each fret is one half step. 12 possible distances before the pattern repeats.
- **Maps**: A set of distances that define a genre sound. Blues map: 0, 3, 5, 6, 7, 10. Rock map: 0, 2, 3, 5, 7, 8, 10.

## Game Modes

1. **Find Home** — Chord gets called, tap the fretboard to find it.
2. **Name the Distance** — Given Home and a target position, work out the distance.
3. **Know Your Maps** — Blues or Rock? Which distances belong where.
4. **Strong Beats** — Right distance on the right beat?
5. **Jam Simulator** — Chord gets called, pick notes for each beat in the bar.
6. **Fretboard Tap** — Given a distance, find it on the neck.

## Play It

Open `grid-system-trainer.html` in any browser, or visit the hosted version on GitHub Pages.

Mobile-friendly. No dependencies. Single HTML file.
